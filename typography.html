Typography

Heading Font: Luckiest Guy

So far, I have chosen Luckiest Guy for my headings. Since it is going to be used for the most visible text on the website I wanted it to have some style, and a good kearning. I want to foucus on college students, so I want to include both genders. I think this font is relax, inspired on the outdoor activity and unisex,so it can attract both genders.

Paragraph Font: Thasadith

It was easier for me to chose Thasadith for the paragraphs of my website than the one for the headings. I tried to use the same context, I wanted a font to be relax and unisex. But also wanted one that could be readable, plus it could actually look diferent, if I decide to add font or cursive if needed in the text.
